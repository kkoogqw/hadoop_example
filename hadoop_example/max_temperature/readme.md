### 1、Requirements
主要实现求每年最大温度的功能，其输出输出如下：

输入：

```
1990 21
1990 18
1991 35
1992 30
1990 25
1992 21
1992 40
```
其中，第一列表示年份，第二列表示温度。


输出：
```
1990 25
1991 35
1992 40
```

### 2、Mapper
在mappper中，以年份为key，温度为value进行存储，不需要进行处理.

### 3、Shuffle
之后Shuffule负责将结果传给Reducer，传送内容为：
```
1990 [21, 18, 25]
1991 [35]
1992 [30, 21, 40]
```
### 4、Reducer
将上述mapper中key对应的value取最大值即可。
